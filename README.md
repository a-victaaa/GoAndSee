# CharaQuest - Team Member Guessing Game

A Java Swing application that plays 20 questions to guess which team member you're thinking of.

## Team Members
- Angelica

## How to Run
1. Clone this repository
2. Open in NetBeans IDE
3. Set main class to `com.charaquest.CharaQuest`
4. Run the project

## Features
- Interactive GUI with 3 screens (Welcome, Game, Result)
- Smart decision tree algorithm
- Answer history tracking
- Win/lose detection
